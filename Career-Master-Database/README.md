# Career Master Database

This folder documents the Career Master Database used as the source of truth for resume content, skills, experience, target roles, career preferences, and job-search workflows.

## Purpose

The Career Master Database serves as the foundation of the AI Career Operations System and supports resume optimization, cover letter generation, job scoring, application tracking, and career planning activities.
